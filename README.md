# 🚀 Kickstarter Projects Dashboard (Power BI)

## 📌 Project Overview

This project analyzes **Kickstarter campaigns** using Power BI to uncover key insights about project performance, funding, and trends across different years and countries.

---

## 📂 Data Source

* Dataset: **Kickstarter Projects**
* Source: Kaggle
* Files Used: Two CSV files (**2016, 2018**)
* Link: https://www.kaggle.com/kemical/kickstarter-projects

---

## 🧹 Data Cleaning & Transformation

The following steps were performed using Power Query:

* Cleaned both datasets individually
* Standardized **data types** across columns
* Removed extra spaces using **Trim**
* Unified column names between both files
* Handled data issues such as formatting errors (e.g., commas)
* Created a new column **File Name** to identify the source file (2016 or 2018)
* Appended (merged) the two datasets into one unified table

---

## 📊 DAX Measures

The following measures were created using DAX:

* **# Projects**
* **# Backers**
* **Total Goal**
* **Total Pledged**

---

## 📈 Key Insights

The dashboard provides the following insights:

* **# Projects by State**
* **# Projects by Year**
* **Total Pledged by Country**
* **# Projects by Main Category**

---

## 🎛️ Interactivity

To enhance usability, the dashboard includes slicers:

* Filter by **Year**
* Filter by **Country**

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)
